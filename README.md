#  Password Generator (Mini Project)

This is a simple **Password Generator web app** built using **JavaScript** and **Tailwind CSS**.  
The app allows users to generate strong and random passwords based on selected options.

This project is created for **practice and learning purposes**.

---

##  Features

- Generate random passwords
- Adjustable password length using slider
- Options to include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Password strength indicator
- Copy password button
- Animated gradient borders
- Custom styled range slider
- Responsive design

---

- Custom CSS

##  How It Works

- Slider controls the password length
- Checkboxes decide which characters to include
- At least one checkbox must be selected
- Password is generated using random character functions
- Password is shuffled for better randomness
- Strength indicator changes color based on selection
